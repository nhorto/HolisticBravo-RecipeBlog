# Holistic Bravo - SEO Keyword Research & Blog Post Strategy

**Date:** February 16, 2026
**Data Source:** Ahrefs keyword explorer (4 exports: vegan-recipes matching/related, vegetarian-recipes matching/related)
**Total Keywords Analyzed:** ~75,000 rows across 4 CSV files
**Filtered to:** KD ≤ 20, Volume ≥ 500 → 122 unique keywords

---

## Executive Summary

From 122 qualifying keywords, we identified **6 priority blog post opportunities** and **3 stretch goals** that align with Holistic Bravo's existing 45-recipe library. The strategy focuses on listicle-style posts ("10 Easy Vegan Dinner Recipes") that each target a keyword cluster and link internally to individual recipe pages.

**Key Finding:** The highest-opportunity keywords are broad category listicles (e.g., "easy vegetarian dinner recipes", "vegan dessert recipes") rather than individual recipe keywords. This is good news - listicle posts naturally accommodate multiple recipes and generate more internal links.

**Estimated Total Traffic Potential:** 50,000-80,000 monthly visits if all 6 priority posts reach page 1.

---

## Methodology

1. Exported keyword data from Ahrefs for seed keywords "vegan recipes" and "vegetarian recipes"
2. Filtered to KD ≤ 20 (realistic for a new site) and Volume ≥ 500
3. Deduplicated across all 4 files (122 unique keywords)
4. Manually matched each keyword to existing recipes by actual relevance (not just keyword overlap)
5. Grouped related keywords into clusters (each cluster = one blog post)
6. Prioritized by: recipe coverage → traffic potential → keyword difficulty

## Keyword Cannibalization Prevention

Each blog post targets a distinct intent. Here's the deconfliction map:

| Post | Primary Intent | Recipes Included |
|------|---------------|-----------------|
| #1 Easy Veg Dinners | "What should I make for dinner tonight?" | Dinner mains only |
| #2 High Protein | "I need protein on a veg diet" | Cross-category (breakfast, dinner, snack) |
| #3 Vegan Desserts | "I want a vegan treat" | Desserts/sweets only |
| #4 Pasta Recipes | "I want to make vegetarian pasta" | Pasta/ravioli/gnocchi only |
| #5 Vegan GF | "I need vegan AND gluten-free" | Cross-category, GF+vegan only |
| #6 Side Dishes | "What sides go with my main?" | Sides/appetizers only |

**Overlap is OK** between posts because the search intent is different. Google won't penalize a tofu pasta appearing in both "easy dinners" and "pasta recipes" - those are different queries answered differently.

## Technical SEO Reminders

1. **URL structure:** `/blog/[keyword-slug]` for blog posts, `/recipe/[recipe-slug]` for recipes
2. **Internal linking:** Each blog post should link to every recipe it mentions using the recipe's page URL
3. **Schema markup:** Blog posts should use `Article` schema; recipe pages should use `Recipe` schema (already planned)
4. **Image alt text:** Include the recipe name in alt text for every image
5. **Meta titles:** `[Primary Keyword] | Holistic Bravo` format, under 60 characters
6. **Meta descriptions:** 150-160 characters, include primary keyword naturally, end with a call to action
