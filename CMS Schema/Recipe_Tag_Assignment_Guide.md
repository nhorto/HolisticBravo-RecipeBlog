# Recipe Tag Assignment Guide

Use this guide to manually assign tags in Framer CMS for each recipe.
Check off each recipe as you complete it.

**Legend:**
- **Categories** = Meal type (Breakfast, Lunch, Dinner, Snack, Dessert, Side Dish, Appetizer, Salad, Beverage)
- **Diet Types** = Dietary classification (Vegetarian, Vegan, Plant-Based, etc.)
- **Cuisines** = Cuisine origin (American, Italian, Asian, etc.)
- **Features** = Attributes (Quick & Easy, No Bake, Budget-Friendly, etc.)
